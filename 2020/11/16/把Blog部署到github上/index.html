把你的博客部署到github上

十一月 16, 2020
  • git
  • hexo

在github上创建一个新仓库随后打开终端

随后进入到博客文件夹到目录里

一直下滑到最后修改 按一下

看到左下角变成insetr就可以修改了

  type: git
  repo: 这个地方放你创建的仓库的链接
  branch: master

随后按一下ESC 看到insetr消失

保存并退出就OK了

随后就输入命令把项目部署到github上

会提示你输入用户名 然后输入密码就OK了。访问你创建仓库时候的域名就OK了。
